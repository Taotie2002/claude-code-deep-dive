# 常见问题（FAQ）

## 关于项目

### Q1: 这个项目是什么？
Claude Code Deep Dive 是一个通过黑盒观测与架构推演来分析 Claude Code 设计原理的技术研究项目。

### Q2: 这个项目有官方授权吗？
**没有**。本项目与 Anthropic 公司无任何关联，不代表官方立场。

### Q3: 项目基于什么做分析？
- 公开信息：Anthropic 官方文档、博客
- 社区讨论：GitHub、Discord、Reddit 等社区的观察和讨论
- 行为测试：通过输入输出测试观察 Claude Code 行为
- 架构推演：基于观察到的行为和行业通用模式推测内部实现

---

## 关于黑盒分析

### Q4: 黑盒推演的结论可靠吗？
**不完全可靠**。本项目采用四级信源标注（L1-L4）来帮助读者判断结论可靠性：
- L1 🟢 实证观测：直接测试验证
- L2 🟡 推断观测：基于源码结构推断
- L3 🔵 社区共识：社区广泛认同
- L4 ⚪ 合理推测：基于行为的逻辑外推

详见：[分析方法论](../claude-code-deep-dive.md#分析方法论)

### Q5: 架构图和流程图是真实源码吗？
**不是**。本报告中的架构图、流程图是为了辅助理解而绘制的，不代表 Claude Code 的真实内部结构。

## 关于Mini Demo

### Q6: Mini Demo是Claude Code的源码吗？
**不是**。Mini Demo是简化复刻版，用于展示核心架构设计思想，**不是**真实源码。

详见：[Mini Demo重要说明](../mini-claude-code/README.md#重要说明这是一个教学演示项目)

### Q7: Mini Demo可以直接用于生产吗？
**不建议**。Mini Demo是教学演示用途，缺少生产级错误处理、安全加固和性能优化。

### Q8: Mini Demo与Claude Code有哪些一致和差异？
详见 Mini Demo README 中的"哪些是推演，哪些是自行设计"表格。
