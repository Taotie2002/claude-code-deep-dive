# 贡献指南

感谢你对 Claude Code 源码解读的兴趣！欢迎以任何方式参与贡献。

---

## 🐛 反馈问题

如果你发现内容有误、源码分析不准确，或有任何建议，欢迎提交 **Issue**：

- 请在标题中简要说明问题
- 描述具体问题或建议
- 如有相关源码片段，请附上引用

## ✍️ 提交内容

欢迎通过 **Pull Request** 贡献内容：

1. **Fork** 本仓库
2. 创建你的分支（`git checkout -b feature/your-topic`）
3. 提交修改（`git commit -m 'Add: your topic'`）
4. 推送到分支（`git push origin feature/your-topic`）
5. 提交 **Pull Request**

## 📋 内容规范

- 使用中文撰写，保持行文风格一致
- 源码引用请注明来源和行号
- 新增章节请同步更新 README.md 目录
- 审计报告请保持客观，结论有据可查

## 💬 其他建议

- 在 GitHub Issue 中讨论大的改动方向
- 对于翻译或纠错类的 PR，可以直接提交
- 实操类章节欢迎补充代码示例

---

期待你的贡献！🚀
