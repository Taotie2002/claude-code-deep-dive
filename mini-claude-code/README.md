# Mini Claude Code - 简化复刻 Demo

> 展示 Claude Code 核心架构的简化实现版本

## ⚠️ 重要说明：这是一个教学演示项目

本 Demo 是**简化复刻版**，用于展示 Claude Code 核心架构的**推测行为模式**。

### 哪些是推演，哪些是自行设计？

| 组件/行为 | 来源 | 说明 |
|-----------|------|------|
| Agent + Tool + Memory 三层架构 | 推演 | 基于Claude Code公开行为分析 |
| Circuit Breaker 熔断机制 | 推演 | 社区广泛观察到的行为 |
| 状态机循环 (think→plan→execute) | 推演 | 基于日志分析 |
| 具体类名/函数名 (Agent, QueryEngine等) | 自行设计 | 为教学清晰而命名，非真实源码 |
| Memory 分层策略 | 推演 | 基于Claude Code公开文档 |

### 为什么不直接复刻真实源码？

1. Claude Code 源码为泄露内容，不可直接引用
2. 本项目的目标是**展示架构设计思想**，而非复刻具体实现
3. 真实源码中的内部API、类名可能会误导学习者

### 适合学习什么？

- ✅ Agent 架构设计模式
- ✅ Tool 调度机制原理
- ✅ Memory 分层管理思想
- ✅ 容错与熔断设计
- ❌ 不适合：学习具体类名、函数签名、源码结构

## 背景

本项目是 Claude Code 源码研究的实战输出，通过简化实现展示：
- 如何从"分析项目"升级为"工程项目"
- 核心 Agent 架构设计
- Tool 调度机制
- Memory 记忆管理

## 项目结构

```
mini-claude-code/
├── README.md           # 本文档
├── agent.py            # 核心 Agent 类（对应 Claude Code src/QueryEngine.ts）
├── tools.py            # Tool 调度机制（对应 Claude Code src/tools.ts）
├── memory.py           # Memory 记忆机制（对应 Claude Code src/memdir/）
├── main.py             # 入口演示
└── requirements.txt    # Python 依赖
```

## 核心架构

### 1. Agent（代理核心）

```
┌─────────────────────────────────────────┐
│              Agent                       │
│  ┌─────────────────────────────────┐    │
│  │  LLM Client (API 调用)           │    │
│  └─────────────────────────────────┘    │
│  ┌─────────────────────────────────┐    │
│  │  Message History (对话历史)       │    │
│  └─────────────────────────────────┘    │
│  ┌─────────────────────────────────┐    │
│  │  Tool Executor (工具执行器)       │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
```

参考源码：
- `src/QueryEngine.ts` - LLM 查询引擎
- `src/Tool.ts` - Tool 类型定义
- `src/tools.ts` - Tool 注册表

### 2. Tool（工具系统）

```
┌─────────────────────────────────────────┐
│           Tool Dispatcher                │
│  ┌─────────────────────────────────┐    │
│  │  Tool Registry (工具注册表)       │    │
│  │  ┌───────┬───────┬───────┐      │    │
│  │  │Bash  │Read   │Write  │...    │    │
│  │  └───────┴───────┴───────┘      │    │
│  └─────────────────────────────────┘    │
│  ┌─────────────────────────────────┐    │
│  │  Permission Check (权限检查)     │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
```

参考源码：`src/tools/` 目录下的各个 Tool 实现

### 3. Memory（记忆系统）

```
┌─────────────────────────────────────────┐
│            Memory System                │
│  ┌─────────────────────────────────┐    │
│  │  Short-term (短期记忆)           │    │
│  │  - 对话历史                       │    │
│  │  - 当前任务上下文                 │    │
│  └─────────────────────────────────┘    │
│  ┌─────────────────────────────────┐    │
│  │  Long-term (长期记忆)            │    │
│  │  - memdir/ 持久化存储            │    │
│  │  - 经验提取                      │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
```

参考源码：
- `src/memdir/` - 记忆目录实现
- `src/services/extractMemories/` - 自动记忆提取

## 与原版 Claude Code 的对应关系

| Mini Claude Code | Claude Code 源码 |
|-----------------|-----------------|
| Agent 类 | `src/QueryEngine.ts` |
| Tool 装饰器 | `src/Tool.ts` + `src/tools.ts` |
| Memory 类 | `src/memdir/` + `src/services/extractMemories/` |
| Tool Executor | `src/tools/` 各个具体工具 |

## 设计亮点

1. **装饰器模式** - 用 `@tool` 装饰器自动注册 Tool
2. **类型提示** - 完整的类型注解
3. **权限模型** - 预留权限检查接口
4. **可扩展性** - 易于添加新 Tool
