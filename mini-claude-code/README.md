# Mini Claude Code - 简化复刻 Demo

> 展示 Claude Code 核心架构的简化实现版本

## 背景

本项目是 Claude Code 源码研究的实战输出，通过简化实现展示：
- 如何从"分析项目"升级为"工程项目"
- 核心 Agent 架构设计
- Tool 调度机制
- Memory 记忆管理

## 项目结构

```
mini-claude-code/
├── README.md           # 本文档
├── agent.py            # 核心 Agent 类（对应 Claude Code src/QueryEngine.ts）
├── tools.py            # Tool 调度机制（对应 Claude Code src/tools.ts）
├── memory.py           # Memory 记忆机制（对应 Claude Code src/memdir/）
├── main.py             # 入口演示
└── requirements.txt    # Python 依赖
```

## 核心架构

### 1. Agent（代理核心）

```
┌─────────────────────────────────────────┐
│              Agent                       │
│  ┌─────────────────────────────────┐    │
│  │  LLM Client (API 调用)           │    │
│  └─────────────────────────────────┘    │
│  ┌─────────────────────────────────┐    │
│  │  Message History (对话历史)       │    │
│  └─────────────────────────────────┘    │
│  ┌─────────────────────────────────┐    │
│  │  Tool Executor (工具执行器)       │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
```

参考源码：
- `src/QueryEngine.ts` - LLM 查询引擎
- `src/Tool.ts` - Tool 类型定义
- `src/tools.ts` - Tool 注册表

### 2. Tool（工具系统）

```
┌─────────────────────────────────────────┐
│           Tool Dispatcher                │
│  ┌─────────────────────────────────┐    │
│  │  Tool Registry (工具注册表)       │    │
│  │  ┌───────┬───────┬───────┐      │    │
│  │  │Bash  │Read   │Write  │...    │    │
│  │  └───────┴───────┴───────┘      │    │
│  └─────────────────────────────────┘    │
│  ┌─────────────────────────────────┐    │
│  │  Permission Check (权限检查)     │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
```

参考源码：`src/tools/` 目录下的各个 Tool 实现

### 3. Memory（记忆系统）

```
┌─────────────────────────────────────────┐
│            Memory System                │
│  ┌─────────────────────────────────┐    │
│  │  Short-term (短期记忆)           │    │
│  │  - 对话历史                       │    │
│  │  - 当前任务上下文                 │    │
│  └─────────────────────────────────┘    │
│  ┌─────────────────────────────────┐    │
│  │  Long-term (长期记忆)            │    │
│  │  - memdir/ 持久化存储            │    │
│  │  - 经验提取                      │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
```

参考源码：
- `src/memdir/` - 记忆目录实现
- `src/services/extractMemories/` - 自动记忆提取

## 快速开始

### 安装依赖

```bash
pip install anthropic
```

### 运行演示

```bash
python main.py
```

## 代码行数统计

| 文件 | 行数 | 说明 |
|------|------|------|
| agent.py | ~150 | Agent 核心逻辑 |
| tools.py | ~100 | Tool 调度 |
| memory.py | ~80 | Memory 管理 |
| main.py | ~70 | 演示入口 |
| **总计** | **~400** | 符合 200-500 行要求 |

## 与原版 Claude Code 的对应关系

| Mini Claude Code | Claude Code 源码 |
|-----------------|-----------------|
| Agent 类 | `src/QueryEngine.ts` |
| Tool 装饰器 | `src/Tool.ts` + `src/tools.ts` |
| Memory 类 | `src/memdir/` + `src/services/extractMemories/` |
| Tool Executor | `src/tools/` 各个具体工具 |

## 设计亮点

1. **装饰器模式** - 用 `@tool` 装饰器自动注册 Tool
2. **类型提示** - 完整的类型注解
3. **权限模型** - 预留权限检查接口
4. **可扩展性** - 易于添加新 Tool
